# XVoice平台版 更新日志

### v3.2.1 (2026年03月11日)

新增: 

1. 登录页面增加清理数据缓存功能. (登录页右上角缩小关闭按钮相邻左侧)
2. 自定义频道增加单个用户音量调节功能. (用户列表鼠标右键指定用户名片)
3. 自定义频道踢出成员增加"踢出"或"踢出并加入黑名单"选项.



修复:

1. 修复自定义频道成员被升职为管理员后, 已升职用户无法对其他成员进行管理操作(禁言/麦, 踢出频道等)
2. 修复退出自定义频道再次进入显示频道不存在问题.
3. 修复解散自定义频道后重新创建崩溃问题.
4. 修复管制员无法点击退出管制问题.
